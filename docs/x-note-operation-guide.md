# X・note運用テンプレート

決算探偵は、グロース・スタンダード・プライムの全市場対応として運用する。

## URLルール

- サイト全体の案内: `https://kessan-tantei.jp/markets`
- 個別企業: `https://kessan-tantei.jp/company/証券コード`
- グロース市場ランキング: `https://kessan-tantei.jp/ranking`
- スタンダード市場ランキング: `https://kessan-tantei.jp/standard/ranking`
- プライム市場ランキング: `https://kessan-tantei.jp/prime/ranking`
- ニュース: `https://kessan-tantei.jp/news`

旧URLや、内容と異なるトップページURLは使わない。
