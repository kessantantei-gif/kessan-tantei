# Phase 7 完了記録

Phase 7 は、会社ページの収益化と比較・分析機能を実運用できる状態にするフェーズです。

## 完了した機能

- 無料版とPro版の表示範囲を分離
- 運営者アカウントとStripe契約者のPro判定を統一
- AI決算サマリー、財務シグナル、決算速報、比較候補のPro全文表示
- スコア根拠、決算探偵の見立て、Red Flags、AI詳細分析、利益・CF推移のPro境界
- 事業テーマ、サブテーマ、明示ライバル、財務類似、成長率類似による比較
- アストロスケール、ispace、QPS研究所、Synspectiveの宇宙関連比較
- 監修済み会社マスタを自動分類より優先
- growth-companiesの全企業へ自動会社マスタを生成
- テーマ別・成長性・収益性・CF・安全性・リスクランキング
- PC・スマートフォンの会社ページ表示順とレスポンシブ対応

## 分類の優先順位

1. 監修済みライバル
2. 同一サブテーマ
3. 同一テーマ
4. 同一33業種
5. 同一17業種
6. 財務・成長率・スコアによる補完

## 運用ルール

- `data/company-master.json` は人が監修する独自データ資産として継続更新する
- 未監修会社は `growth-companies.json` の会社名・業種情報から自動分類する
- 監修済み分類は自動分類より常に優先する
- 比較結果は投資助言ではなく、決算データの理解補助として表示する

## 監査

```bash
npm run audit:phase7
```

監査対象:

- 全企業の会社マスタ生成
- テーマ・サブテーマ・ビジネスモデルの欠損
- 監修済みライバルの銘柄コード
- 宇宙関連比較の必須企業
- 主要ランキングの存在
- 全ProコンポーネントのPro判定統一
